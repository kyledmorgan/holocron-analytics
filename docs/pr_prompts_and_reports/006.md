- provides a **high-level ‚Äúbig picture‚Äù** of the repo documentation,
- provides **deep links** to documents and, where possible, **section anchors** within documents,
- is **additive** and does **not** duplicate the full content of each doc,

We also want to update our **agent guidance** so that whenever a user asks to ‚Äúupdate documentation,‚Äù the agent follows a standard process:
1) locate the documentation index,  
2) review relevant docs,
